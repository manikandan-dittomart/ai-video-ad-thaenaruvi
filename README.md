# AI Video Advertisement – THAENARUVI Honey

## 🚀 Project Overview
This project demonstrates an end-to-end AI-powered workflow to create a cinematic product advertisement video for **THAENARUVI Honey**.

The entire ad was built using AI tools — from prompt-based image generation to video creation and final editing with audio.

---

## 🎯 Workflow

Prompt → Image Generation → Frame-to-Video → Final Ad Creation

---

## 🖼️ Image Generation

Three key frames were generated using structured prompts:

1. **Frame 1 – Scene Introduction**
   - Honeybee flying through a flower garden toward a kitchen window

2. **Frame 2 – Brand & Mascot**
   - Animated honeybee with THAENARUVI honey jar

3. **Frame 3 – Product Highlight**
   - Ultra-realistic cinematic honey jar product shot

📂 Prompts available in: `/prompts/`

---

## 🎬 Frame-to-Video (Veo 3.1)

Image sequences were converted into video using cinematic prompts:

- Image 1 → Image 2 transition
- Image 2 → Image 3 transition

📂 Video prompts available in: `/video_prompts/`

---

## 🎞️ Final Video

The generated frames were converted into a complete advertisement video.

📂 Final output video available in:
- `/video/` folder *(or repository root)*

---

## 🔊 Audio

Voice-over audio was added to enhance storytelling and advertisement quality.

📂 Audio file:
- `voiceover.mp3` *(or /audio/voiceover.mp3)*

---

## 🧠 Tools Used

- **Banana Pro 2** – Image Generation  
- **Veo 3.1** – Frame-to-Video Creation  
- **iMovie** – Video Editing & Audio Integration  

---

## 💡 Key Learnings

- AI can replicate a full advertisement production pipeline  
- Prompt engineering plays a critical role in output quality  
- Combining multiple AI tools creates powerful results  
- Storytelling + AI = High-quality content creation
